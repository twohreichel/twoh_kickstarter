.. _what-it-does:

What does it do?
================

The **TWOH Kickstarter** extension serves as a comprehensive starting point for
new TYPO3 projects. It bundles essential extensions and configurations to help
developers quickly set up professional TYPO3 websites with:

*  **Performance optimizations** through static file caching and source optimization
*  **SEO-ready setup** with Yoast SEO integration
*  **Search functionality** via Apache Solr integration
*  **Image optimization** through TinyPNG compression
*  **Best practice TypoScript** following TYPO3 standards

.. _features:

Features
========

Quick Project Setup
-------------------

Get your TYPO3 project running in minutes with pre-configured TypoScript
templates and sensible default settings.

Performance Optimizations
-------------------------

Built-in caching and optimization tools including:

*  Static file cache for maximum performance
*  HTML source code optimization
*  Minification support

SEO Ready
---------

Pre-configured SEO settings with:

*  Yoast SEO integration
*  XML sitemap generation
*  Meta tag management
*  Open Graph support

Solr Search
-----------

Apache Solr integration out of the box for powerful search functionality.

Image Optimization
------------------

Automatic image compression with TinyPNG integration to reduce page load times.
