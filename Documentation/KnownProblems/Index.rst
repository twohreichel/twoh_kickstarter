==============
Known Problems
==============

This section lists known issues and their workarounds.

.. _known-problems-none:

Currently known issues
======================

No critical issues are currently known. If you encounter any problems,
please report them on the `GitHub issue tracker <https://github.com/twohreichel/twoh_kickstarter/issues>`__.

.. _known-problems-compatibility:

Compatibility Notes
===================

TYPO3 Version
-------------

This extension is tested and compatible with TYPO3 13.4.x. It is not compatible
with TYPO3 12.x or earlier versions.

PHP Version
-----------

*  PHP 8.2: Fully supported
*  PHP 8.3: Fully supported
*  PHP 8.4: May work but not officially tested yet

Extension Conflicts
-------------------

There are no known conflicts with other extensions. If you experience any
conflicts, please report them.
